Background

The data for the visualization comes from .

The CENSUS_STATEAGESEX_BINS file contains data that consists of three dimensions: STATE, SEX, and AGE and two measures POP2000 and POP2008. This data has been aggregated across the AGE dimension for each STATE and SEX. Here's an example:

      {
        "Alabama": {
          "m": [
            {
              "range": [
                0,
                10
              ],
              "pop2000": 345925,
              "pop2008": 346855
            },
            {
              "range": [
                11,
                21
              ],
              "pop2000": 359490,
              "pop2008": 355269
            },
            {
              "range": [
                22,
                32
              ],
              "pop2000": 321476,
              "pop2008": 339535
            }
          ]
        }
      }

For instance, this means that Alabama had 345,925 boys in the age range of 0-10 years in 2000 and 346,855 boys in the age range of 0-10 years in 2008.

The visualization is created in Javascript with the help of the jQuery library and Underscore templates.
